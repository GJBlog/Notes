#### 查看Pod版本

```bash
pod --version
```
#### 查看Gem源

```bash
gem sources -l
```

#### 更新Pod

```bash****
sudo gem install cocoapods
```

#### 更新指定版本的Pod

```
sudo gem install -n /usr/local/bin cocoapods -v 1.5.3
```
