## CSS-Sprite

#### 什么是CSS Sprite

* 一种CSS图像合成技术，将各种小图片合并到一张图片上，然后利用CSS的背景定位来显示对应的图片部分
* 有人翻译为：CSS雪碧、CSS精灵

#### 使用CSS Sprite的好处

* 减少网页的http请求数量，加快网页响应速度，减轻服务器压力
* 减少图片总大小
* 解决了图片命名的困扰，只需要针对一张集合的图片命名
* 更换风格方便，只需要在少数张图片上修改图片的颜色或样式，整个网页的风格就可以改变
