## 配置图床

1、推荐使用GitHub的方式

* 使用CDN加速访问速度
* 使用PicGo来上传图床

访问的图片路径：

路径 = 前缀(https://cdn.jsdelivr.net/gh/) + 仓库名称 + file的文件夹路径 + fileName

```
https://cdn.jsdelivr.net/gh/GJBlog/Notes/科学方式/images/science_00.jpg
```
