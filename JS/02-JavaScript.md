# 02-JavaScript
JavaScript 类型、变量
### 2.1 
1. 区分大小写: 关键字、变量、函数名和所有标识符都区分大小写，HTML中的属性名和标签并不区分大小写，在JavaScript中必须是小写
2. 直接量: 程序中直接使用的数据值
3. 标识符: 以字母、下划线(_)、美元($)开始，后续可以是数字、字母、下划线、美元
4. 数据类型分为：原始类型(数字、字符串(不可变)、布尔值、null、undefined)和对象类型(属性的集合，特殊对象：全局对象)
5. 类是对象类型的子类型，除了数组类、函数类，还有Date类、正则类、错误类
6. JavaScript解释器的内存管理机制，垃圾回收机制
7. 变量无类型
8. 不区分整数值和浮点数值,全局变量Infinity(正无穷大)、NaN(非数字值)、-Infinity(负无穷大)
9. 字符串是一组由16位值组成的不可变的有序序列
