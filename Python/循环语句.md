## 循环语句

#### while、for 也可以跟else进行搭配使用

* 当循环正常结束(非break结束)后，会执行1次else语句

```python
i = 0
while i < 5:
    print(i)
    i += 1
else:
    print('done')
```

结果：0 1 2 3 4 done

```python
for i in range(5):
    print(i)
else:
    print('done')
```
结果：0 1 2 3 4 done

```python
i = 0
while i < 5:
    print(i)
    if i == 3:
        break
    i += 1
else:
    print('done')
```
结果：0 1 2 3 

```python
for i in range(5):
    print(i)
    if i == 3:
        break
else:

```

结果：0 1 2 3
